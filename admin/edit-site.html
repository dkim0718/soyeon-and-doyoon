🎨 사이트 디자인

웨딩 홈페이지(영문+한국어)의 글꼴과 색상입니다. 저장하면 모든 방문자에게 적용됩니다. 글꼴 이름은 Google Fonts 기준. 비워둔 칸은 기본값을 유지하고, 사이트 타이틀에 None 을 입력하면 제목 글꼴을 따라갑니다.

색상 팔레트 (두 사이트 공통)

영어(EN) / 한국어(KR) 웨딩 홈페이지의 이야기·일정·Q&A 등을 코드 수정 없이 편집합니다. 저장하면 이 브라우저에서 즉시 반영됩니다. 모든 방문자에게 공유하려면 Supabase 연결이 필요합니다 (현재는 편집한 기기에서만 보입니다).
